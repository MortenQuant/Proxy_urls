# Offshore Wind Project Portal Registry — UK NSIP

Consent register URLs and proxy fallback URLs for UK offshore wind DCO projects on the Planning Inspectorate NSIP register.

Last updated: 2026-04-16

## How to use

When web_fetch fails on an NSIP URL (HTTP 429), use the proxy URL:
1. web_search the proxy URL as the query string
2. web_fetch the same proxy URL
